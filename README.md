# About

Comcalc was created to automate billing for Community Kinesiologists working with ICBC and WSBC patients.  

Feel free to checkout the app here: comcalc.netlify.app

# Tech Stack

Front-end: React

Back-end: NodeJS, Tom Tom API, Google Maps API, Twilio API

# Features

1. Calculator for mileage and travel time for ICBC and WSBC client locations for a single day

2. ICBC session end date reminder 

3. Automated appointment reminder via text message
